# Taco Town Truck

## Description
Website displays current menu of available items, allows users to add items to their order and supports order submissions.

[System design documentation](https://whimsical.com/taco-truck-KYJFzR5ecQkVby5RLCL74v)

[Taco Truck API](https://app.swaggerhub.com/apis-docs/Detroit_Labs/Taco_Truck/1.0.0)

## Getting Started

### Dependencies
* Node version 16.18 or higher

### Executing program

To run locally (defaults to port 3000):
```
npm run start
```
